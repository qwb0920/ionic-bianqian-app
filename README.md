# Ionic 2 便签 Application

使用 typescript+angualr2+ionic2



## 开始

npm isntall   之前要安装了（npm install -g ionic）

ionic serve

最后补充一句  基于nodejs  v4.x.x版本，不然起不来服务，

ps：可以装个nvm这样就可以nodejs多版本共存了。
